Pigment lookup

Use this database of pigments to look up a pigment and the corresponding color from different paint manufacturers. For example, the pigment code PY65 is generically known as Arylide yellow and Hanza yellow. Da Vinci Paint Co. sells a single-pigment paint color with the PY65 pigment called Arylide Yellow Deep whereas Daniel Smith calls its version Hansa Yellow Deep.

You can also look up a particular color from a paint manufacturer so you can find the comparable color from a different paint brand. For example, if you're used to using Cobalt Blue Deep from Da Vinci Paint Co. and want to try Daniel Smith paints, you can use this database to find the corresponding paint from Daniel Smith. Since Cobalt Blue Deep is made from the pigment PB28, you can use that information to learn that Daniel Smith calls its version of this color Cobalt Blue.


The label usually lists the pigment code

NOTE: Many pigments can produce variations on a hue, such as the pigment PB28. For example, Da Vinci Paint Co. produces both Cobalt Blue (aqua blue) and Cobalt Blue Deep (mid blue) from PB28. This database will allow you to see these variations so you can more precisely choose the correct color for you.
